== Description ==

Add some buttons to tiny admin editor, buttons for item WoW (poor, normal, commun, rare, epic, ...) and Youtube buttons


Automatic generate some shortcodes for Youtube (version 2 and 3), automatic generates shortcode for colorisation items for Blizzard/wowhead/Magelo items. Enqueue option for one of this Tooltips (Magelo or wowhead).


some class to use with html tag 

Color items

* poor
* normal
* commun
* rare
* epic
* legend
* artefact

Some others

* success
* info
* error
* blizzquote


You may use this outside the admin editor buttons 

e.g.

`<p class="epic">Your text here....</p>`


== Installation ==

1. Upload 'tinywowcolor' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Define few attributs and selector on page option > Appearence > Tiny WoW Youtube.
